Canto Clients simple and effective client logo shortcode.

== Description ==

Canto Clients simple and effective client logo shortcode. It's interrogated with BxSlider.

= Shortcode =
	[canto_clients]

 = Attributes List: =
  * count (int): Amount of logos at view.

	`Default: 4`

  * hover (bool): If 1 (true), logos will be black and white. When hover event will be happened, logo will be colorful.

    `Default: 1`

    `Options: '0 - False', '1 - True'`

= Install the Plugin =

 Full instruction of this plugin installation

 e.g.

 2. Activate the plugin through the 'Plugins' menu in WordPress
 3. Now Add client logos and add shortcode at widget, page and post.
